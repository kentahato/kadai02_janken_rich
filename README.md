# ①課題名
スロットマシーンアプリ
## ②課題内容（どんな作品か）
・ボタン（３つ）を押すと各スロットの目が表示されるマシーン。
・ゾロ目を出すと成功。
## ③アプリのデプロイURL
デプロイしている場合はURLを記入（任意）
## ④アプリのログイン用IDまたはPassword（ある場合）
・無し
## ⑤工夫した点・こだわった点
・最初（ボタンを押す前）、スロットが回っている風に動きを付けた点。
・ゾロ目が揃うとcongratulations画面が表示される点。
・画面右上の☆マークをクリックするとスリーセブンが揃うモードに変わる点。
## ⑥難しかった点・次回トライしたいこと（又は機能）
・各スロットの目を表示する際、html/cssの表示で実現するべきか、画像の表示で実現するべきか迷いました。
